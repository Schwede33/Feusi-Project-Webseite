## Getting Started

als Erstes klone dieses Repository lokal auf deinen Rechner

git clone (https://github.com/Schwede33/Feusi-Project-Webseite/)


cd w3schools-database  # Wechselt in den neuen Ordner w3schools-database


git pull  # update repository d
ocker-compose up -d  # (wenn es probleme gibt, system neustarten)

code .  # Startet Visual Studio Code im aktuellen Ordner

Wechsel in der Command App in den Web-App Ordner

cd web-app

und dann gibst du 

npm run dev

ein.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
